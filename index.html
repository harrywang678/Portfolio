What are some Projects that I completed?

Mini-CPU

For this project, I designed and implemented a basic CPU using Logisim-Evolution, creating a sequential datapath that includes essential components like instruction memory, a register file, an ALU, data memory, and a program counter. I also designed my own instruction set architecture and developed an assembler program to translate assembly code into machine code. After assembling a demo program, I successfully simulated the CPU’s operation. I documented the entire process, including a user manual for my CPU and instructions for using the assembler.

Trivia Game

This project involves developing a Trivia game where a server manages the game and multiple clients (players) connect to participate. The server handles tasks such as setting up connections, reading questions from a database, accepting player connections, and managing the gameplay. The clients connect to the server, provide their names, and participate in the game by answering trivia questions. The server keeps track of scores and determines the winner at the end of the game. Both server and client programs involve command-line argument parsing, socket programming, and multiplexed I/O.
